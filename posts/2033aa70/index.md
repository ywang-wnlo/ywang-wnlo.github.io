## 长度

常用的长度单位

| 单位 | 含义                  | 换算成 pt | 换成算成 mm |
| ---- | --------------------- | --------- | ----------- |
| pt   | 基本单位              | 1 pt      | 0.35146 mm  |
| mm   | 毫米                  | 2.84 pt   | 1 mm        |
| cm   | 厘米                  | 28.4 pt   | 10 mm       |
| in   | 英寸                  | 72.27 pt  | 0.35146 mm  |
| ex   | 当前字体的 x 字母高度 | -         | -           |
| em   | 当前字体的 m 字母宽度 | -         | -           |

## 数学公式

### 段落中（隐式）

三种均可，任意选择，以 E=mc<sup>2</sup> 为例

```tex
\(E=mc^2\)
$E=mc^2$
\begin{math}E=mc^2\end{math}
```

### 单独成段（显式）

三种均可，任意选择

```tex
\[E=mc^2\]
\begin{displaymath}E=mc^2\end{displaymath}
\begin{equation}E=mc^2\end{equation}
```

## 居中，左对齐，右对齐

### 居中

```tex
\begin{center}
  balabala
\end{center}
```

### 左对齐

```tex
\begin{flushleft}
  balabala
\end{flushleft}
```

### 右对齐

```tex
\begin{flushright}
  balabala
\end{flushright}
```

## 参考文献配置

\bibliography{bibfile}
